# Kulüp Belgeleri

Bu klasör, kulübün resmi belgelerini içerir.

## Tüzük Belgesi

Kulüp tüzüğünü bu klasöre aşağıdaki isimle ekleyin:

```
yeni-kadikoy-spor-kulubu-tuzuk.pdf
```

Bu dosya, `/hakkimizda/tuzuk` sayfasında indirilebilir olacaktır.

## Dosya Ekleme

1. PDF dosyasını bu klasöre kopyalayın
2. Dosya adının tam olarak `yeni-kadikoy-spor-kulubu-tuzuk.pdf` olduğundan emin olun
3. Dosya boyutunun makul olduğundan emin olun (önerilen: < 5MB)

## Notlar

- Dosya adında Türkçe karakter kullanmayın
- Dosya adında boşluk yerine tire (-) kullanın
- PDF formatında olmalıdır
